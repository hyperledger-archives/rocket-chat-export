
1. Creating Crypto Materials
    $ cryptogen generate --config=./crypto-config.yaml
        org0
        org1
        org2
        org3
   
    Output: crypto-config directory will be created will all peers and orderer


2. Creating Orderer Genesis Block
    $ configtxgen -profile FourOrgsOrdererGenesis -outputBlock genesis.block

    Output: genesis.block file will be generated
